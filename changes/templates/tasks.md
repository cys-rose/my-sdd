## 1. 使用规则

1. 本文件是 `/apply` 阶段的执行合同。
2. 没有对应 `spec.md` 时，不得执行。
3. 每个 Task 必须是原子变更，建议控制在 3-5 个文件。
4. 不得跨 Task 偷跑；若发现范围变化，先回写 `spec.md`。
5. 每个 Task 完成后必须记录验证证据与文档同步情况。
